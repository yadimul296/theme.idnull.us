    Code Video Untuk Postingan seperti demo di atas biasanya player video kaya gini suka di pakai video untuk movies

    
<div id="myElement">
Loading the player ...</div>
<script type="text/javascript">
var playerInstance = jwplayer("myElement")
playerInstance.setup({
    width: "100%",
    aspectratio: "16:9",
    image: "IMAGES_LINK",
    sources: [{
         file: "VIDEO_LINK",
       }],
         tracks: [{
                file: "SUBTITLE_LINK",
                label: "Indonesian",
                kind: "captions",
               "default": true
         }]
  });
</script>


    • IMAGE_LINK : gantilah dengan URL/Link Gambar atau Link Posting anda.
    • VIDEO_LINK : gantilah dengan URL/Link Video anda. di sini juga bisa menggunakan Video youtube
    • contoh URL https://www.youtube.com/watch?v=IdejtDN9gyc tinggal tambahin aja pada VIDEO_LINK
    • SUBTITLE_LINK : gantilah dengan URL/Link Subtitle.srt link anda. itu juga jika anda ingin menambahkan subtitle pada video tersebut
    • Selesai
